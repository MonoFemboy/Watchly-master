# Security Policy

## Supported Versions

Versions that will get security updates:

| Version | Supported          |
| ------- | ------------------ |
| 5.0.x   | :white_check_mark: |
| 4.0.x   | :x:                |
| < 4.0   | :x:                |

## Reporting a Vulnerability

Create an issue ticket if there is a Vulnerability and the Contributors will attempt to fix the vulnerability.
